Это врущий сервер точного времени, он врет на время указанное в файле lietime. 
Для работы нужно запустить python servervremeny.py и отправить ему любой запрос.
Для тестирования прекреплен файл testserver,им можно проверить работоспособность сервера запустив сначала сам сервер 
python servervremeny.py, а затем тестовый файл python testserver.py
